# M3U Player Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a basic macOS desktop M3U/IPTV player: load a playlist from a URL or file, browse by group, search across all channels, favorite channels, remember favorites/last-watched, and play the selected MPEG-TS stream in an embedded VLC video pane.

**Architecture:** Pure, testable core modules (`playlist`, `store`, `source`) with no UI/network side effects in their logic, plus two thin isolation layers for the external dependencies (`player` wraps python-vlc; `source` wraps network fetch with cache fallback). A PySide6 three-pane window (`main`) wires them together. Favorites are keyed by `group + name` so they survive credential-token rotation in stream URLs.

**Tech Stack:** Python 3, PySide6 (Qt), python-vlc (uses installed VLC.app engine), stdlib `urllib`.

## File Structure

- `m3u_player/__init__.py` — package marker
- `m3u_player/playlist.py` — `Channel` dataclass + `parse_m3u(text) -> list[Channel]`. Pure.
- `m3u_player/store.py` — `Config` load/save (JSON in App Support); favorite matching. Pure logic + file IO.
- `m3u_player/source.py` — `load_playlist_text(source, cache_path, fetcher)` — file read or URL fetch with cache fallback.
- `m3u_player/player.py` — thin python-vlc wrapper: play/stop/pause/volume/attach/fullscreen.
- `m3u_player/main.py` — PySide6 three-pane window + app entry point.
- `tests/test_playlist.py`, `tests/test_store.py`, `tests/test_source.py` — unit tests for the pure modules.
- `requirements.txt`, `README.md`, `run.sh` — setup/run.

TDD applies to `playlist`, `store`, `source` (pure, deterministic). `player.py` and `main.py` are integration layers verified by launching the app (documented in README / final task), since they require VLC and a display.

```python
import re
from dataclasses import dataclass

_GROUP_RE = re.compile(r'group-title="([^"]*)"')
_STREAMID_RE = re.compile(r'/(\d+)\.[A-Za-z0-9]+(?:\?.*)?$')


@dataclass(frozen=True)
class Channel:
    name: str
    group: str
    url: str
    stream_id: str


def _stream_id(url: str) -> str:
    m = _STREAMID_RE.search(url)
    return m.group(1) if m else ""


def parse_m3u(text: str) -> list[Channel]:
    channels: list[Channel] = []
    pending = None  # (name, group) awaiting a URL line
    for raw in text.splitlines():
        line = raw.strip()
        if not line:
            continue
        if line.startswith("#EXTINF"):
            group_match = _GROUP_RE.search(line)
            group = group_match.group(1).strip() if group_match else ""
            name = line.split(",", 1)[1].strip() if "," in line else ""
            pending = (name or "Unnamed", group or "Ungrouped")
        elif line.startswith("#"):
            continue  # other directives (#EXTM3U, #EXTGRP, etc.)
        else:
            if pending is not None:
                name, group = pending
                channels.append(Channel(name, group, line, _stream_id(line)))
                pending = None
    return channels
```

```python
import json
from pathlib import Path
from m3u_player.playlist import Channel


def default_config_path() -> Path:
    base = Path.home() / "Library" / "Application Support" / "M3UPlayer"
    return base / "config.json"


def _fav_key(group: str, name: str) -> str:
    # Stable across credential-token rotation: group + name only.
    return f"{group}␟{name}"


class Config:
    def __init__(self, path: Path | None = None):
        self.path = path or default_config_path()
        self.source: dict | None = None            # {"type": "file"|"url", "value": str}
        self.favorites: list[dict] = []            # [{group, name, stream_id}]
        self.last_group: str | None = None
        self.last_watched: dict | None = None      # {group, name, stream_id}

    # --- favorites ---
    def is_favorite(self, c: Channel) -> bool:
        key = _fav_key(c.group, c.name)
        return any(_fav_key(f["group"], f["name"]) == key for f in self.favorites)

    def toggle_favorite(self, c: Channel) -> None:
        key = _fav_key(c.group, c.name)
        for i, f in enumerate(self.favorites):
            if _fav_key(f["group"], f["name"]) == key:
                del self.favorites[i]
                return
        self.favorites.append({"group": c.group, "name": c.name, "stream_id": c.stream_id})

    # --- persistence ---
    def to_dict(self) -> dict:
        return {
            "source": self.source,
            "favorites": self.favorites,
            "last_group": self.last_group,
            "last_watched": self.last_watched,
        }

    def save(self) -> None:
        self.path.parent.mkdir(parents=True, exist_ok=True)
        self.path.write_text(json.dumps(self.to_dict(), indent=2))

    @classmethod
    def load(cls, path: Path | None = None) -> "Config":
        cfg = cls(path=path)
        if cfg.path.exists():
            data = json.loads(cfg.path.read_text())
            cfg.source = data.get("source")
            cfg.favorites = data.get("favorites", [])
            cfg.last_group = data.get("last_group")
            cfg.last_watched = data.get("last_watched")
        return cfg
```

from pathlib import Path
from urllib.request import urlopen


def _default_fetcher(url: str) -> str:
    with urlopen(url, timeout=30) as resp:  # noqa: S310 (user-supplied provider URL)
        return resp.read().decode("utf-8", errors="replace")


def load_playlist_text(source: dict, cache_path: Path, fetcher=_default_fetcher) -> str:
    cache_path = Path(cache_path)
    if source["type"] == "file":
        return Path(source["value"]).read_text(encoding="utf-8", errors="replace")

    # URL source: try to fetch fresh, cache it; on failure fall back to cache.
    try:
        text = fetcher(source["value"])
    except Exception:
        if cache_path.exists():
            return cache_path.read_text(encoding="utf-8", errors="replace")
        raise
    cache_path.parent.mkdir(parents=True, exist_ok=True)
    cache_path.write_text(text, encoding="utf-8")
    return text
```

import sys
import vlc


class Player:
    """Thin wrapper over python-vlc bound to a Qt widget's window handle."""

    def __init__(self):
        self._instance = vlc.Instance("--no-xlib", "--quiet")
        self._mp = self._instance.media_player_new()

    def bind_window(self, widget) -> None:
        wid = int(widget.winId())
        if sys.platform == "darwin":
            self._mp.set_nsobject(wid)
        elif sys.platform.startswith("win"):
            self._mp.set_hwnd(wid)
        else:
            self._mp.set_xwindow(wid)

    def play(self, url: str) -> None:
        media = self._instance.media_new(url)
        self._mp.set_media(media)
        self._mp.play()

    def stop(self) -> None:
        self._mp.stop()

    def pause(self) -> None:
        self._mp.pause()

    def set_volume(self, value: int) -> None:
        self._mp.audio_set_volume(int(value))

    def toggle_fullscreen(self) -> None:
        self._mp.toggle_fullscreen()
```

## Task 5: Three-pane UI + wiring

**Files:**
- Create: `m3u_player/main.py`

Integration layer (Qt). Verified by launching. Requirements captured as behavior below.

**Behavior contract:**
- Window titled "M3U Player", ~1100x700, three horizontal panes via `QSplitter`.
- **Left (Groups):** `QListWidget`. First row is "★ Favorites". Remaining rows are sorted unique group names from the parsed channels. Selecting a group repopulates the middle list. Selecting "★ Favorites" shows all favorited channels across groups.
- **Middle (Channels):** a `QLineEdit` search box on top + a `QListView` backed by a custom `QAbstractListModel` (`ChannelListModel`) holding the currently displayed `list[Channel]`. A model handles 26k rows cheaply. Typing in the search box (with a ~200ms `QTimer` debounce) filters across **all** channels case-insensitively by name and replaces the model's list; clearing the box restores the selected group's channels. Favorited channels show a "★ " prefix in their display text.
- **Right (Video):** a black `QWidget` (`video_frame`) that VLC binds to, plus a control bar: Play/Pause button, Stop button, volume `QSlider` (0–100, default 80), and a "⛶" fullscreen button. A label above/over the frame shows the now-playing channel name. Double-clicking the video toggles fullscreen.
- **Toolbar/top bar:** "Open…" (file or URL) and "Refresh" actions. "Open…" shows a small dialog offering a URL text field or a file picker.
- **Playback:** double-clicking a channel row, or selecting it and clicking Play, calls `player.play(channel.url)`, updates the now-playing label, sets `config.last_watched`, and saves config. Switching channels reuses the same video pane.
- **Favorites:** right-click a channel row → context menu "Add to favorites"/"Remove from favorites" → `config.toggle_favorite(channel)` + `config.save()` + refresh the row's star and the Favorites group if visible.
- **Startup:** load `Config`. If `config.source` set, load playlist text via `source.load_playlist_text` (cache path = `~/Library/Application Support/M3UPlayer/playlist_cache.m3u`) in a background `QThread`/`QThreadPool` so the UI shows a "Loading…" state and stays responsive; parse; populate groups; restore `last_group` selection and highlight `last_watched` (do **not** auto-play). If no source, prompt with the Open dialog. On fetch failure with no cache, show a non-blocking message.
- **Errors:** wrap `player.play` and playlist loading so failures show a status-bar/message ("Couldn't play this channel" / "Couldn't load playlist") without crashing.
- **Entry point:** `def main():` builds `QApplication`, shows the window, `sys.exit(app.exec())`. Guard with `if __name__ == "__main__": main()`.
